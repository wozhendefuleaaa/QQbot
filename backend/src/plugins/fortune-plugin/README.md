# 今日运势插件

一个独立的今日运势查询插件，支持文字版运势查询和每日悔签功能。

## 功能特点

- 📅 **每日运势查询** - 支持 `#今日运势` 或 `#运势` 命令
- 🔄 **悔签功能** - 支持 `#悔签` 或 `#重新抽取运势` 每日重抽一次
- 📝 **文字版回复** - 无需外部图片依赖，响应速度快
- 💾 **数据持久化** - 使用本地 JSON 文件存储每日记录与悔签状态
- 🌟 **多运势等级** - 包括超吉、大吉、吉、中吉、小吉、凶、大凶
- ⭐ **星级评价** - 每个运势等级都有对应的星级显示
- 📱 **多场景支持** - 支持群聊和私聊场景

## 命令说明

| 命令 | 别名 | 功能 |
|------|------|------|
| `#今日运势` | `#运势` | 查询当日运势 |
| `#悔签` | `#重新抽取运势` | 每日重抽一次运势 |

## 运势格式

```
@用户
您的今日运势为：
[运势等级]
[星级评价]

<运势描述>
[详细解释]

图片源自网络|如有侵权|请反馈删除
仅供娱乐|相信科学|请勿迷信
```

## 安装说明

1. 将 `fortune-plugin` 目录复制到 `backend/src/plugins/` 目录
2. 重启机器人服务或通过 API 重载插件
3. 插件会自动加载并注册命令

## 数据存储

- 运势数据存储在 `backend/src/plugins/fortune-plugin/data/fortune.json`
- 包含用户每日运势记录和悔签状态
- 自动创建数据目录和文件

## 技术特点

- 纯文字回复，响应速度快
- 本地数据持久化，无需外部依赖
- 支持群聊和私聊场景
- 命令别名支持，使用更便捷
- 冷却时间设置，防止命令刷屏
- 完整的错误处理和日志记录
